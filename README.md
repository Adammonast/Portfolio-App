## Installation

OS X:

```sh
yarn install
```

Windows:

```sh
npm install
```

## Development setup

After you fork/clone the repository to our local machine and use the installation command: run one of the following start commands to open the project on your localhost

OS X:

```sh
yarn start
```

Windows:

```sh
npm start
```
